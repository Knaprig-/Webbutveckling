My Works

Here are pictures and brief descriptions of some projects that I have completed in recent years. If you're interested in knowing more about any of the projects, feel free to contact me!

Battleships - AS3 project

This was a project where I in Action Script 3 (a programming language) had to recreate the game Battleships. This version is single player only, and you simply try to get a high score by finding all of the randomly placed ships as soon as possible. The game tracks your hits, misses and number of different ship types you've sunk. When the ships are placed at the start of the game, they can't be placed directly next to each other. Knowing this, the game is not fully random as you can rule out some positions of the ships.
